# 观测指标与判定标准
所有观测仅记录**客观现象**，统一描述口径，方便数据汇总。

## 一、四大观测指标（表格内统一填写）
1. 光斑偏移程度：无偏移 / 轻微偏移 / 明显偏移
2. 衍射条纹状态：无条纹 / 浅淡条纹 / 清晰明显条纹
3. 光斑展宽：基本不变 / 轻微展宽 / 显著展宽
4. 光强变化：无衰减 / 轻微衰减 / 明显衰减

## 二、各档位参考现象（供对照）
1. 间隙 0.1 mm：偏移、条纹、展宽均非常显著；
2. 间隙 0.25 mm（核心档）：效应最强，是本次实验重点观测状态；
3. 间隙 0.4 mm：效应逐步减弱，条纹变淡、光斑收缩；
4. 间隙 0.5 mm：衍射、偏移基本消失，光束接近原始准直状态。

## 三、图像拍摄要求
1. 拍摄角度正对观测屏，画面完整包含整个光斑；
2. 每张图片命名规则：`间隙数值_材质_日期`
   示例：`0.25mm_金属片_20260608`
3. 同一档位至少留存2张实拍图。

## 四、无效数据判定
出现以下情况，数据作废，重新测试：
1. 激光直接接触遮挡片，发生反射、遮挡；
2. 气流剧烈扰动，光斑持续抖动无法稳定；
3. 间隙档位与标准偏差过大。
